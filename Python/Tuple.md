### [문제 6] 함수의 다중 반환값(Multiple Return Values) 처리

```python

    pi = 3.14159
    circumference = 2 * pi * radius
    area = pi * (radius ** 2)
    return circumference, area  # 튜플로 반환됨

# 함수를 호출하면서 둘레(c)와 넓이(a)를 각각 독립된 변수로 동시에 언패킹하여 받으세요.
________ = calculate_circle(5)

print(f"둘레: {c}, 넓이: {a}")

```
