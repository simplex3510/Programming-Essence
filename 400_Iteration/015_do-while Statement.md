# `do-while` Statement

반복문 중 `do-while`에 대하여 다룬다.

`do-while`문은 `while`과 마찬가지로 특정한 조건을 만족하는 동안 코드를 반복하기 위한 반복문이다.  
또한 의도한다면 무한 반복(infinite loop)도 가능하다.  

이때, `do` 구문은 반드시 한 번은 실행된다.

`do-while`문의 문법은 다음과 같다.

```cs
do
{
    반복할 코드
} while (반복 조건식);
```

`do-while`문은 내부 코드를 우선 실행시킨 후, 그 다음에 조건식을 검사한다.

예를 들어, 어떤 FPS 게임에서 어떤 아이템이 있다고 가정하자.  
해당 아이템은, 1번 사격할 때 3발의 투사체가 발사된다고 한다.

이를 `while`문으로 구현하면 다음과 같다.

```cs
int count = 0;
const int PROJECTILE_COUNT = 3;

do
{
    (투사체 발사);
    count++;
} while (count < PROJECTILE_COUNT);
```

위 `while`문은 다음과 같은 순서로 작동된다.  

1.  **내부 코드 실행**  
    `do-while`문 내부의 코드를 실행한다.

2. **반복 조건 검사**  
    `count`의 값이 `PROJECTILE_COUNT`보다 작은지 검사한다.  
    반복 조건이 참일 경우, 내부 코드의 가장 위쪽으로 실행 순서를 옮긴다.  
    반복 조건이 거짓일 경우, 반복문을 탈출한다.  

3. **반복**  
    1번 단계로 실행 순서를 옮겨, 반복문을 진행한다.

즉, 위 코드는 `count`가 `0`에서 `3`까지 증가하여 총 3번의 투사체를 발사한다.  
하지만 조건 검사는 `count`가 `1`, `2`, `3`일 때 실행된다.
마지막 검사에서 `count`가 `PROJECTILE_COUNT`보다 작지 않으므로 투사체 발사 로직이 실행되지 않고 반복문을 탈출하게 된다.
