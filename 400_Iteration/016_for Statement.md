# `for` Statement

반복문 중의 하나인 `for`문에 대해서 다룬다.

`for`문은 특정 코드를 정해진 횟수만큼 반복하기 위해 구상되어 만들어졌다.

`for`문의 문법은 다음과 같다.

```cs
for (반복 변수 초기화; 반복 조건식; 증감문)
{
    반복할 코드
}
```

## for문의 작동 방식

예를 들어, 어떤 FPS 게임에서 어떤 아이템이 있다고 가정하자.  
해당 아이템은, 1번 사격할 때 3발의 투사체가 발사된다고 한다.

이를 반복문으로 구현하면 다음과 같다.

```cs
const int PROJECTILE_COUNT = 3;

for (int count = 0; count < PROJECTILE_COUNT; count++)
{
    (투사체 발사);
}
```

위 `for`문은 다음과 같은 순서로 작동된다.

1. **반복 변수 초기화**  
    `count`에 `0`이 대입되며, `for`문 진입 시 최초 1회만 실행된다.

2. **반복 조건 검사**  
    `count`의 값이 `PROJECTILE_COUNT`보다 작은지 검사한다.  
    반복 조건이 참일 경우, 내부에 있는 투사체 발사 로직을 실행한다.  
    반복 조건이 거짓일 경우, 반복문을 탈출한다.

3. **반복 변수의 증가**  
    반복문 내부의 코드를 실행한 후에 반복 변수를 증가시킨다.  
    즉, `count`가 후위 증가(`++`)이므로, 1을 증가시킨다.

4. **반복**  
    2번 단계로 다시 돌아가서, 반복문을 진행한다.

즉, 위 코드는 `count`가 `0`에서 `1`, `1`에서 `2`, `2`에서 `3`까지 증가하여 총 3번의 투사체를 발사한다.  
이때, `count`가 `PROJECTILE_COUNT`보다 작지 않으므로 투사체 발사 로직이 실행되지 않고 반복문을 탈출하게 된다.

## Nested-for statement (중첩 for문)

`for`문 내부에 `for`문을 다시 사용하는 중첩된 구조를 가질 수도 있다.  
이러한 중첩 `for`문은 여러 상황에서 자주 사용되는 경우가 많다.

일반적으로는 2개의 `for`문을 사용한 '이중 `for`문'을 가장 많이 사용한다.

예를 들어, 어떤 게임에서 그리드에 맞게 건물을 지을 수 있는 시스템이 있다고 가정하자.  
이때 건물을 짓기 위해서는, 우선적으로 그리드가 정의되어 있어야 한다.  
즉, 건물을 지을 수 있는 단위의 영역이 필요하다.

가령, 다음과 같은 코드로 나타낼 수가 있다.

```cs
int sizeRow = (가로 영역의 크기를 입력받음);
int sizeCol = (세로 영역의 크기를 입력받음);

for (int y = 0, y < sizeCol; y++)
{
    for (int x = 0; x < sizeRow; x++)
    {
        (그리드 1칸 출력);
    }

    (다음 줄로 이동);
}
```
