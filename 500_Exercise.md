# 500_Iteration_Exercise

반복문과 배열을 활용한 실습 문제를 다룬다.

## C: 반도체 및 하드웨어 시스템 제어

하드웨어 제어에서는 데이터를 순차적으로 처리하거나 센서 값을 관리하는 능력이 필수적이다.

1. **센서 데이터 평균 필터**  
    10개의 전압 센서 값을 배열에 저장하고, `for`문을 사용하여 전체 평균 전압을 계산하세요. (실수형 나눗셈 활용)

2. **불량 칩 검수**  
    50개의 반도체 수율 데이터(`0` 또는 `1`)가 저장된 배열에서 `0`(불량)의 개수를 찾아 출력하고, 불량률을 % 단위로 계산하세요.

3. **순차적 LED 점등**  
    8개의 LED가 배열 형태로 연결되어 있습니다. `for`문을 이용해 1번부터 8번까지 순차적으로 켜지는 시뮬레이션을 작성하세요. (인덱스 활용)

4. **ADC(Analog-to-Digital) 변환**  
    0~5V 사이의 센서 값 5개를 배열로 입력받아, 이를 0~1023 사이의 정수 값으로 변환하여 새 배열에 저장하세요.

5. **온도 임계값 경고**  
    24시간 동안 측정된 온도 배열을 검사하여, 80도 이상인 구간이 발생하면 "냉각 시스템 가동"을 출력하고 해당 시간(인덱스)을 모두 표시하세요.

6. **메모리 주소 시뮬레이터**  
    10칸짜리 배열을 메모리 공간이라 가정하고, 특정 인덱스에 데이터를 저장하거나 비우는 메뉴 방식을 `for`문과 `if`문으로 구현하세요.

7. **최대 전력 소모 구간 찾기**  
    시간대별 전력 소모량이 저장된 배열에서 가장 전력을 많이 소비한 시간과 그 값을 찾으세요.

8. **패리티 비트(Parity Bit) 확인**  
    7개의 이진 데이터가 담긴 배열을 확인하여 1의 개수가 홀수이면 "오류 발생", 짝수이면 "정상"을 출력하세요.

9. **데이터 신호 평탄화**
    이전 값과 현재 값의 차이가 너무 크면(노이즈) 이를 보정하는 로직을 배열 순회 루프 내에서 구현하세요.

10. **반도체 웨이퍼 등급 분류**  
    10개의 웨이퍼 품질 점수 배열을 순회하며 90점 이상은 A, 80점 이상은 B, 나머지는 C등급으로 분류하여 각 등급의 개수를 세세요.

## C#: 게임 로직 및 시스템 설계

게임 개발에서는 캐릭터의 능력치 관리, 인벤토리, 다수의 몬스터 처리에 배열과 반복문이 핵심이다.

1. **몬스터 웨이브 HP 관리**  
    5마리 몬스터의 HP를 배열로 선언하고, `for`문을 사용해 모든 몬스터에게 20의 데미지를 입힌 후 남은 HP를 출력하세요.

2. **인벤토리 아이템 검색**  
    10칸의 인벤토리 배열(`string[]`)에서 "포션"이 있는지 `for`문으로 확인하고, 있다면 몇 번째 칸에 있는지 출력하세요.

3. **스테이지별 경험치 테이블**  
    1레벨부터 10레벨까지 필요한 경험치가 직전 레벨의 1.2배씩 증가합니다. `for`문을 사용하여 레벨별 필요 경험치 배열을 완성하세요.

4. **연타 공격 데미지 계산**  
    사용자가 5번의 공격을 시도합니다. 각 공격의 데미지를 배열에 저장하고, 전체 데미지의 합계와 가장 강력했던 일격(Max 값)을 찾으세요.

5. **캐릭터 선택 시스템**  
    4명의 캐릭터 이름이 담긴 배열을 출력하고, 사용자로부터 번호를 입력받아 선택된 캐릭터의 이름을 출력하세요. (인덱스 범위 체크 포함)

6. **팀 평균 전투력**  
    3인 파티의 각 캐릭터 전투력을 입력받아 배열에 저장하고, 파티의 평균 전투력이 500 이상이면 "던전 입장 가능"을 출력하세요.

7. **강화 성공 기록**  
    10번의 강화 시도 결과(`true`/`false`)를 배열에 담고, 연속으로 성공한 최대 횟수를 `for`문과 `if`문으로 계산하세요.

8. **상점 물가 변동**  
    아이템 가격이 담긴 배열을 `for`문으로 순회하며, 이벤트 기간을 맞아 모든 아이템 가격을 10% 할인된 가격(형변환 주의)으로 갱신하세요.

9. **랭킹 보드 업데이트**  
    기존 Top 5 점수 배열이 있을 때, 새로운 점수가 5위 안에 진입할 수 있는지 판별하는 로직을 작성하세요.

10. **1차원 맵 이동 시뮬레이션**  
    10칸짜리 배열(`int[]`)에서 플레이어의 위치를 `1`로 표시합니다. 사용자 입력을 받아 플레이어의 위치를 왼쪽이나 오른쪽으로 한 칸씩 이동시키세요. (배열 경계 검사 필수)
