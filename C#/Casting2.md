# 🎮 C# 상속 형변환(Casting, is, as) 문제 5선

### 1. 탈것 시스템과 안전한 기능 실행 (`as` 활용)

게임 내의 다양한 탈것(Vehicle)을 관리하는 시스템입니다. 안전한 다운캐스팅을 통해 특정 탈것의 고유 기능을 실행하세요.

* **조건:**
* 부모 클래스 `Vehicle`을 만들고, 이를 상속받는 자식 클래스 `Horse`(말)와 `Airship`(비행선)을 만드세요.
* `Horse` 클래스에는 `public void Gallop()` 메서드가, `Airship` 클래스에는 `public void Fly()` 메서드가 독자적으로 존재합니다.
* `Main` 메서드에서 `Vehicle[] garage = new Vehicle[2];` 배열을 만들고 `Horse`와 `Airship` 객체를 하나씩 할당하세요.
* `for`문을 통해 배열을 순회하며, **`as` 연산자를 사용하여 형변환**을 시도하세요.
* 형변환 결과가 `null`이 아님을 확인(`if`문 활용)한 뒤, 각각의 고유 메서드(`Gallop`, `Fly`)를 호출하는 코드를 작성하세요.

### 2. 마법사의 광역 스킬과 타겟 분류 (`is` 활용)

마법사가 광역 공격을 수행할 때, 범위 내의 대상이 아군인지 적군인지 판별하여 다른 효과를 적용해야 합니다.

* **조건:**
* 부모 클래스 `Target`을 만들고, 이를 상속받는 자식 클래스 `Ally`(아군)와 `Enemy`(적군)를 만드세요.
* `Ally` 클래스에는 `public void Heal()` 메서드가, `Enemy` 클래스에는 `public void TakeDamage()` 메서드가 존재합니다.
* `Main` 메서드에서 `Target[] areaTargets = new Target[3];` 배열을 만들어 `Ally`, `Enemy`, `Ally` 객체를 무작위로 채우세요.
* `for`문과 **`is` 연산자를 활용**하여 대상의 타입을 확인한 뒤, 아군에게는 힐을, 적군에게는 데미지를 입히는 분기 로직을 완성하세요.

### 3. 강제 형변환(Casting)의 한계와 예외 방지 (논리 분석)

런타임 에러의 원인을 메모리 관점에서 추론하는 서술형/코드 수정 문제입니다.

```csharp
class Pet { }
class Dog : Pet { }
class Cat : Pet { }

class Program
{
    static void Main()
    {
        Pet myPet = new Cat();
        
        // 아래 코드는 컴파일은 되지만 실행 시 InvalidCastException 에러가 발생합니다.
        Dog myDog = (Dog)myPet; 
    }
}

```

* **문제:**
1. `(Dog)myPet` 처럼 괄호를 사용하는 명시적 형변환(Casting)의 위험성을 설명하세요.
2. 이 프로그램이 강제 종료되지 않고 안전하게 실행될 수 있도록 `is` 또는 `as` 연산자를 사용하여 코드를 올바르게 수정하세요.

### 4. 부모 타입 매개변수와 다운캐스팅 (상점 거래 시스템)

하나의 메서드로 다양한 아이템의 구매 영수증을 처리하는 상점 시스템을 구현합니다.

* **조건:**
* 부모 클래스 `Item`을 만들고, 이를 상속받는 `Weapon` 클래스와 `Armor` 클래스를 만드세요.
* `Shop` 클래스를 만들고 내부에 `public void PrintReceipt(Item purchasedItem)` 메서드를 정의하세요.
* 이 메서드는 매개변수로 모든 `Item`을 받지만, 내부에서는 다음과 같이 판별합니다.
* 구매한 아이템이 `Weapon` 타입이면 -> "무기를 구매했습니다." 출력
* 구매한 아이템이 `Armor` 타입이면 -> "방어구를 구매했습니다." 출력


* **제한 사항:** 메서드 오버로딩을 사용하지 말고, 오직 하나의 `PrintReceipt` 메서드 안에서 **형변환 연산자(`is` 또는 `as`)만을 사용**해 구현하세요.

### 5. 최상위 `object` 배열의 데이터 분류 (종합)

C#의 최상위 부모인 `object` 배열을 활용하여, 서로 다른 데이터 타입을 분류하고 본래의 기능을 복원하는 문제입니다.

* **조건:**
* `object[] 혼합배열 = new object[3];`을 선언하세요.
* 0번 칸에는 정수 `25` (플레이어 나이), 1번 칸에는 문자열 `"Game Over"`, 2번 칸에는 앞서 만든 `Horse` 객체를 대입하세요.
* `for`문을 돌며 이 배열의 요소를 하나씩 꺼내어, **타입을 판별한 뒤 원래의 타입으로 다운캐스팅**하여 다음과 같이 처리하세요.
* 정수(int)인 경우: "나이: 25세" 출력
* 문자열(string)인 경우: 대문자로 변환(`ToUpper()` 메서드 활용)하여 "GAME OVER" 출력
* `Horse` 객체인 경우: 내부의 `Gallop()` 메서드 호출
