# 🎮 C# System.Array, this, base 개념 강화 문제 (중상급)

### 1. `System.Array` 메서드를 활용한 랭킹 보드 정렬

게임이 끝난 후 플레이어들의 점수를 정렬하고 역순으로 배치하여 탑 랭커를 가려내는 시스템을 구현하세요.

* **조건:**
* `int[] scores = { 450, 1200, 890, 310, 1500 };` 배열을 선언하세요.
* `System.Array` 클래스가 제공하는 정적 메서드(`static method`)들을 활용하여 다음을 수행하세요.
1. 점수를 오름차순으로 정렬하세요.
2. 정렬된 배열을 뒤집어(역순) 내림차순(높은 점수가 맨 앞으로)이 되도록 만드세요.
3. `System.Array.Clear()` 메서드를 사용하여 하위 2개의 점수(배열의 마지막 두 칸)를 `0`으로 초기화하세요.


* 처리 단계마다 배열의 상태를 출력하여 값이 어떻게 변하는지 확인하세요.

### 2. `this` 키워드를 이용한 생성자 체이닝 (Constructor Chaining)

매개변수의 개수에 따라 여러 생성자를 만들 때, 중복 코드를 줄이고 `this`를 통해 생성자를 연결하는 구조를 설계합니다.

* **조건:**
* `Character` 클래스를 생성하고 `string name`, `int level`, `int hp` 필드를 선언하세요.
* 다음의 구조대로 **생성자 3개**를 만드세요.
1. 매개변수 3개`(name, level, hp)`를 모두 받아 필드를 초기화하는 생성자.
2. 매개변수 1개`(name)`만 받는 생성자 -> **`this` 키워드를 사용하여** 1번 생성자를 호출하되, 레벨은 `1`, HP는 `100`을 기본값으로 넘겨줍니다.
3. 매개변수가 없는 기본 생성자 -> **`this` 키워드를 사용하여** 2번 생성자를 호출하되, 이름은 `"신규유저"`를 넘겨줍니다.


* `Main`에서 기본 생성자로 객체를 만들고 모든 필드가 정상적으로 초기화되었는지 출력하세요.

### 3. `this` 참조를 통한 '나 자신'을 매개변수로 전달하기

전투 시스템에서 캐릭터가 버프 상점이나 다른 시스템에 **자기 자신의 인스턴스 참조**를 통째로 넘겨주는 논리를 구현합니다.

* **조건:**
* `BuffManager` 클래스를 만들고, 내부에 `public void ApplyAtkBuff(Player player)` 메서드를 정의하세요. 이 메서드는 전달받은 플레이어의 `atk`(공격력)를 50만큼 증가시킵니다.
* `Player` 클래스는 `public int atk = 100;` 필드를 가집니다.
* `Player` 클래스 내부에 `public void ActivateSkill(BuffManager manager)` 메서드를 만드세요.
* **핵심:** `ActivateSkill` 메서드 내부에서 `manager.ApplyAtkBuff(...)`를 호출할 때, **`this` 키워드를 매개변수로 넘겨주어** 자기 자신을 버프 매니저에게 전달하도록 구현하세요.

### 4. `base` 키워드를 통한 부모 생성자 호출과 필드 구별

부모 클래스와 자식 클래스에 동일한 이름의 변수가 있거나, 자식 생성자가 실행되기 전 부모의 필수 데이터를 먼저 세팅해야 할 때 `base`가 어떻게 쓰이는지 검증합니다.

* **조건:**
* 부모 클래스 `Monster`를 만들고, `string grade` 필드를 선언하세요. 생성자에서 이 `grade`를 입력받아 초기화하도록 만드세요.
* 자식 클래스 `Boss`를 만들고 `Monster`를 상속받게 하세요. `Boss` 내부에도 동일하게 `string grade;` 필드를 선언하세요 (부모의 필드를 가리는 상태).
* `Boss` 생성자에서 `string baseGrade`(부모용)와 `string bossGrade`(자식용)를 매개변수로 받으세요.
* **`base(...)` 문법을 사용**하여 `baseGrade`를 부모 생성자에게 넘겨주고, 생성자 본문 안에서는 **`this.grade`와 `base.grade`를 활용**하여 두 필드를 각각 알맞게 초기화하세요.

### 5. `System.Array` 확장과 상속 구조에서의 데이터 탐색

부모 타입 배열에 담긴 자식 객체들을 `System.Array` 메서드를 활용해 탐색하고 제어하는 종합 문제입니다.

* **조건:**
* 부모 클래스 `Item`을 상속받는 `Weapon` 클래스를 만드세요. `Weapon` 클래스는 `int durability`(내구도) 필드를 가집니다.
* `Main`에서 `Item[] inventory = new Item[4];` 배열을 생성하고, `Weapon` 객체 4개를 각각 내구도 `80, 40, 100, 20`으로 설정하여 채우세요 (업캐스팅).
* 배열 내의 `Weapon` 중 **내구도가 50 미만인 첫 번째 무기**가 몇 번째 인덱스에 있는지 찾아내고 그 무기의 내구도를 100으로 수리(`durability = 100`)하는 로직을 구현하세요.
