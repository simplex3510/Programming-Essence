# SWITCH-Statement (`switch`문)

`switch`문은 `if`문과 유사하게 조건을 검사하여 조건에 맞는 경우에만 코드를 실행할 수 있다.

`if`문의 조건식에서는 다양한 논리 및 비교(관계) 연산자가 사용되지만, `swtich`문에서는 `==`를 전제하고 있다는 점이 다르다.

다음은 `swtich`문의 문법과 구조를 나타내고 있다.

```cs
switch ([비교 대상])
{
    case [상수 라벨1]:
        // [비교 대상] == [상수 라벨1]이 true일 경우 실행되는 코드
        break;
    case [상수 라벨2]:
        // [비교 대상] == [상수 라벨1]이 true일 경우 실행되는 코드
        break;
    case [상수 라벨3]:
        // [비교 대상] == [상수 라벨3]이 true일 경우 실행되는 코드
        break;
    default:
        // [비교 대상]과 일치하는 라벨이 없을 때 실행되는 코드
        break;
}
```

예를들어 다음과 같은 코드가 있다.

```cs
int num = 3;

if (num == 1)
{
    // codes...
}
else if (num == 2)
{
    // codes...
}
else if (num == 3)
{
    // codes...
}
else
{
    // codes...
}
```

위 코드는 동일한 변수에 대해 여러 개의 `else if`를 나열하는 경우다.  
이러한 경우에 `switch`문을 사용하면 더 효율적이고 가독성이 좋은 코드를 작성할 수 있다.

```cs
int num = 3;

switch (num)
{
    case 1:
        // codes...
        break;
    case 2:
        // codes...
        break;
    case 3:
        // codes...
        break;
    default:
        // codes...
        break;
}
```

`if`문은 조건을 검사하며 `false`일 경우에 다음 조건을 검사하는 순차적인 방식이다.  
반면, `switch`문은 비교 대상과 일치하는 `case`문으로 곧바로 이동하여 실행 순서를 건너뛰는 점이 다르다.

이렇듯 실행의 흐름이 순차적이지 않게끔 다른 곳으로 이동하는 것을 **점프(jump)**라고 한다.

코드 실행 도중에 `break`를 만나면, `switch`문을 빠져나가게 된다.
이렇게 `case`마다의 코드 실행의 끝을 명시해야 한다.

만약 명시하지 않는다면 다음과 같은 코드가 된다.

```cs
int num = 3;

switch (num)
{
    case 1:
        // codes1...
    case 2:
        // codes2...
    case 3:
        // codes3...
    default:
        // codes4...
}
```

이러한 코드는 `num`이 `1`인 경우에 `case 1:`으로 점프하고, 그 아래의 모든 코드를 실행하며, 결국 `switch`문의 끝에 도달하여 탈출하게 된다.
만약 `num`이 `2`라면, 마찬가지로 `case 2:`로 점프하고, 위의 경우를 반복하게 된다.

이렇게 `switch`문에서 특정 `case`의 실행이 끝난 뒤, 멈추지 않고 바로 이후 있는 `case`의 코드까지 이어서 실행되는 현상을 '**Fall-through**'라고 한다.

즉, `break`를 만나야지만 코드의 흐름을 `switch` 밖으로 탈출 시킬 수 있다.  
따라서 `break`를 반드시 명시해주어 점프 후의 코드 실행 범위를 명확하게 지정해 주어야 한다.

하지만 `break`를 명시하는 강제성은 언어에 따라 다르다.

C#의 경우에는 언어적 차원에서 모든 `case`문 이후에는 `break`를 필수적으로 명시해야 하고, 그렇지 않을 경우 컴파일 에러를 발생시킨다.  
하지만 C/C++의 경우에는 그러한 강제성이 수반되지는 않으며, 컴파일 경고에 그친다.
