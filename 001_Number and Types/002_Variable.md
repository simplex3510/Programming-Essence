# Variable (변수)
프로그래밍에서의 변수는 하나의 값을 저장할 수 있는 메모리 공간이며, 이 메모리 공간에 대한 이름을 부여한 것이다.

변수를 만들 때, 보통 3가지를 필요로 한다.
```cs
// [Type] [Name] = [Value];

int damage = 120;
int defense = 80;
```

변수는 한 번 저장해두면 여러 번 재사용 할 수 있다. <br>
또한, 프로그램 실행 중 값이 계속 변할 수 있다. <br>

가독성에도 중요한 영향을 미친다. <br>
매직 넘버가 아닌, 해당 값에 이름을 붙여줌으로써 의도와 의미를 명확하게 할 수 있다.
```cs
const float PI = 3.14159f;  // const 변수 = 상수
string UserName = "Kim doe";
```
이를 통해 코드의 흐름과 맥락을 이해하는데 큰 도움이 된다.

## Declaration (선언)
변수의 선언은 다음과 같은 형태다.
```cs
float healthPoint;
```

변수의 선언이란, 변수 또는 상수의 타입과 이름을 컴퓨터에게 알려주는 행위다. <br>
즉, 컴퓨터에게 자료형의 크기만큼 메모리를 확보하게 하고, 이 확보된 메모리를 지칭할 때의 이름을 '선언'하는 것이다.

## Definition (정의), Assignment (대입), Initialize (초기화)
대입 또는 초기화는 변수, 상수의 실제 값을 확보된 메모리에 저장하는 것을 의미한다. <br>
즉, 선언된 변수에 값을 대입(Assignment)하여 저장하는 과정이다.
```cs
healthPoint = 100.0f;
```

또한 선언과 동시에 초기화(Initialize)도 가능하다.
```cs
float healthPoint = 100.0f;
```

이렇게 값을 할당하는 과정 전반을 '정의(Definition)'라고도 한다.
