# Variable (변수)
프로그래밍에서의 변수는 하나의 값을 저장할 수 있는 메모리 공간이며, 이 메모리 공간에 대한 이름을 부여한 것이다.

변수를 만들 때, 보통 3가지를 필요로 한다.
```cs
// [Type] [Name] = [Value];

int damage = 120;
int defense = 80;
```

변수는 한 번 저장해두면 여러 번 재사용 할 수 있다. <br>
또한, 프로그램 실행 중 값이 계속 변할 수 있다.
```cs
int gold = 500;
gold = 250;
```

문자, 문자열 또한 변수에 저장될 수 있는 값이다.
```cs
char skill1 = 'Q'
string UserName = "Kim Doe";
```


변수는 가독성에도 중요한 영향을 미친다. <br>
매직 넘버가 아닌, 해당 값에 이름을 붙여줌으로써 의도와 의미를 명확하게 할 수 있다.
```cs
const float PI = 3.14159f;  // const 변수 = 상수형 변수 = 상수
```
이를 통해 코드의 흐름과 맥락을 이해하는데 큰 도움이 된다.

## Declaration (선언)
변수의 선언은 다음과 같은 형태다.
```cs
float healthPoint;
```

변수의 선언이란, 변수 또는 상수의 타입과 이름을 컴퓨터에게 알려주는 행위다. <br>
즉, 컴퓨터에게 자료형의 크기만큼 메모리를 확보하게 하고, 이 확보된 메모리를 지칭할 때의 이름을 '선언'하는 것이다.

## Definition (정의), Assignment (대입), Initialize (초기화)
대입 또는 초기화는 변수, 상수의 실제 값을 확보된 메모리에 저장하는 것을 의미한다. <br>
즉, 선언된 변수에 값을 대입(Assignment)하여 저장하는 과정이다.
```cs
healthPoint = 100.0f;
```

또한 선언과 동시에 초기화(Initialize)도 가능하다.
```cs
float healthPoint = 100.0f;
```

이렇게 값을 할당하는 과정 전반을 '정의(Definition)'라고도 한다.

## Naming Rule (명명법)
변수의 이름을 짓는 것에 강제적인 규칙은 없다. <br>
하지만, 협업 환경이나 혹은 미래의 자신이 어떤 코드를 보고 이해할 수 있을 정도의 가독성과 전달력이 있어야 한다. <br>

C#을 기준으로, 일반적으로 변수는 다음과 같은 규칙을 따른다.
* 변수는 의미하고자 한 바를 명확하게 표현할 수 있는 **명사형 단어**를 사용한다.
    ```cs
    int age = 26;
    int cost = 30;
    ```
* 여러 개의 명사가 조합되어 사용된다면, \*카멜 표기법 (Camel Case)로 표기한다.
    > **카멜 표기법 (Camel Case)** <br>
    여러 단어를 연달아 사용할 때 각 단어의 첫 글자를 대문자로 적되, 맨 앞에 오는 글자는 소문자로 표기하는 명명법.

    ```cs
    float healthPoint = 100.0f;
    float coolDownTime = 2.75f;
    ```

## 상수형 변수, `const` 키워드
`const`(constant) 키워드로 수식된 변수는 상수형 변수로, **상수**라고도 한다. <br>
기본적으로 한번 초기화 된 변수의 값을 바꾸지 않는 경우에 사용된다. <br>
C#의 경우에 상수를 선언할 때 초기화가 동반되어야 한다.

C#을 기준으로, 일반적으로 상수는 다음과 같은 규칙을 따른다.
* 상수는 의미하고자 한 바를 명확하게 표현할 수 있는 **명사형 단어**를 사용한다.
* 여러 개의 명사가 조합되어 사용된다면, 각 단어 사이에 `_`(underscore) 기호를 사용한다.
    ```cs
    const float MAX_MANA_CAP = 100.0f;
    MAX_MANA_CAP = 0.0f;  // 컴파일 에러 발생
    ```

## mutable(가변), immutable(불변)
**변수**는 처음부터 **가변**적임을 전제로 하기에, `mutable`이라고 불리기도 한다. <br>
그렇다면, **상수**는 그 값이 **불변**하기에 `immutalbe`이라고도 한다.
