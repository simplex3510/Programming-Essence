# Relational Operator (관계 연산자)
관계 연산자는 좌항과 우항, 두 값을 비교하여 그 결과가 참(`true`/`1`)인지 거짓(`false`/`0`)인지를 판별하는 연산자를 의미한다.

## 종류 및 의미

관계 연산자의 종류와 의미를 설명하기 위한 전제로 다음의 코드가 있음을 가정한다.

```cs
int a = 10;
int b = 20;
```

다음 표는 관계 연산자의 종류와 의미를 설명한다.

| 연산자 | 의미 | 예시 | 결과 |
| :--: | :-- | :--: | --: |
| `==` | 좌항과 우항의 값이 **같다** | `a == b` | `false` (`0`) |
| `!=` | 좌항과 우항의 값이 **같지 않다** | `a != b` | `true`(`1`) |
| `>`  | 좌항이 우항보다 **크다** | `a > b` | `false` (`0`) |
| `<`  | 좌항이 우항보다 **작다** | `a < b` | `true`(`1`) |
| `>=` | 좌항이 우항보다 **크거나 같다** | `a >= b` | `false` (`0`) |
| `<=` | 좌항이 우항보다 **작거나 같다** | `a <= b` | `true`(`1`) |
