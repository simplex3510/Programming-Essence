# Assignment Operator (대입 연산자)

대입 연산자는 연산자의 왼쪽에 있는 피연산자에 연산자의 오른쪽에 있는 피연산자의 값을 저장한다.

기호는 `=`를 사용하며, 문법은 다음과 같다.
```txt
[left-operand] = [right-operand]
```

사용법은 대표적으로 다음과 같다.
```cs
int num1 = 10;
int num2 = num1 + 20;
int num3 = num1 + num2;
```

# Compound Assignment Operator (복합 대입 연산자)

대입 연산자는 일부 다른 연산자와 조합이 가능하다.  
산술 연산자, 또는 비트 연산자와 조합이 가능하다.

## 복합 : 산술 연산자

산술 연산자와 결합된 형태의 복합 대입 연산자는 다음과 같다.  

* `+=`: 오른쪽의 피연산자를 왼쪽의 피연산자에 **더한 후 저장**한다.
* `-=`: 오른쪽의 피연산자를 왼쪽의 피연산자에 **뺀 후 저장**한다.
* `*=`: 오른쪽의 피연산자를 왼쪽의 피연산자에 **곱한 후 저장**한다.
* `/=`: 오른쪽의 피연산자를 왼쪽의 피연산자를 **나눈 후 몫을 저장**한다.
* `%=`: 오른쪽의 피연산자를 왼쪽의 피연산자를 **나눈 후 나머지를 저장**한다.

```cs
int num1 = 10;

num1 += 14;  // num1 = num1 + 14;
num1 -= 20;  // num1 = num1 - 20;
num1 *= 2;   // num1 = num1 * 2;
num1 /= 2;   // num1 = num1 / 2;
num1 %= 3;   // num1 = num1 % 3;
```

## 복합 : 비트 연산자

비트 연산자와 결합된 형태의 복합 대입 연산자는 다음과 같다.  

* `&=`: 왼쪽 피연산자와 오른쪽 피연산자를 **비트 논리곱(AND) 연산 후 저장**한다.
* `|=`: 왼쪽 피연산자와 오른쪽 피연산자를 **비트 논리합(OR) 연산 후 저장**한다.
* `^=`: 왼쪽 피연산자와 오른쪽 피연산자를 **비트 배타적 논리합(XOR) 연산 후 저장**한다.
* `<<=`: 왼쪽 피연산자의 비트를 오른쪽 피연산자만큼 **왼쪽으로 시프트(Shift) 후 저장**한다.
* `>>=`: 왼쪽 피연산자의 비트를 오른쪽 피연산자만큼 **오른쪽으로 시프트(Shift) 후 저장**한다.

```cs
int num2 = 5;    // 2진수: 0101

num2 &= 3;   // num2 = num2 & 3;  (0101 & 0011 -> 0001, 즉 1)
num2 |= 2;   // num2 = num2 | 2;  (0001 | 0010 -> 0011, 즉 3)
num2 ^= 1;   // num2 = num2 ^ 1;  (0011 ^ 0001 -> 0010, 즉 2)
num2 <<= 2;  // num2 = num2 << 2; (0010 -> 1000, 즉 8)
num2 >>= 1;  // num2 = num2 >> 1; (1000 -> 0100, 즉 4)
```
