# Logical Operator (논리 연산자)

논리 연산자는 주로 관계(비교) 연산자로 만들어진 여러 개의 조건식을 하나로 결합하거나, 조건의 참, 거짓을 반전시킬 때 사용한다.

다음 코드를 전제로 하여, 논리 연산자의 종류와 의미는 다음과 같다.
```cs
int num1 = 10;
int num2 = 20;
```
| 연산자 | 의미 | 예시 | 결과 |
| :--: | :-- | :-- | :-- |
| `&&` | 좌항과 우항이 모두 `true`인가? | `(num1 == 10) && (num2 != 20)` | `false` |
| `\|\|` | 좌항과 우항이 하나라도 `true`인가? | `(num1 == 10) \|\| (num2 != 20)` | `true` |
| `!` | 우항의 불리언 결과를 반전시킨다. | `!(num1 == 10)` | `false` |
