# Conditional Operator (조건 연산자)

조건 연산자는 `if`/`else`문을 한 줄로 간결하게 표현하는 연산자다.

피연산자가 3개이므로 '**삼항 연산자(Ternary Operator)**'라고도 불린다.

문법 구조는 다음과 같다.
```cs
// [조건식] ? [true일 때의 반환값] : [false일 때의 반환값]
```

두 수를 비교하여 더 큰 값을 판단하는 코드가 다음과 같다고 가정한다.
```cs
uint num1 = 10;
uint num2 = 20;

uint max = 0;
if (num1 > num2)
{
   max = num1;
}
else
{
   max = num2;
}
```

이를 조건 연산자로 표현하면 다음과 같다.
```cs
uint num1 = 10;
uint num2 = 20;

uint max = (num1 > num2) ? num1 : num2;
```

이렇듯, 간단한 비교를 진행할 때에 한하여 사용된다.

중첩된 `if`를 표현하고자 조건 연산자를 중첩해서 사용할 경우 가독성이 현저히 떨어지고, 실수 발생의 확률 또한 증가하게 된다.
```cs
// [조건식1] ? : 값1 : [조건식2] ? 값2 : 값3;
```

따라서 삼항 연산자는 제한적으로 사용하는 것이 좋고 남용하지 말아야 한다.
