# Programming Essence

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

**Programming Essence**는 C/C++과 C#을 중심으로 프로그래밍의 본질과 언어 및 문법 개념을 체계적으로 학습하고 정리하기 위한 리포지토리입니다.  

모든 내용은 마크다운(`문서`) 형식으로 관리됩니다.

## 📖 학습 목표
* **C/C++, C# 언어의 깊이 있는 이해**  
    단순 문법 암기가 아닌 작동 원리와 철학을 다룹니다.
* **프로그래밍 본질 탐구**  
    메모리 구조, 설계 원칙, 디자인 패턴 등 언어를 관통하는 핵심 개념을 학습합니다.
* **지식의 기록과 공유**  
    학습한 내용을 논리적으로 정리하여 지속 가능한 지식 베이스를 구축합니다.

## 🤝 기여 방법 (Contribution)

이 리포지토리는 여러분의 참여로 더욱 정확하고 풍성해집니다. 잘못된 개념이나 오타 제보, 내용 보완 등 모든 형태의 기여를 환영합니다!

* **Issue 제보**  
    문서의 오류, 오타, 혹은 보완이 필요한 개념을 발견하셨다면 [Issues] 탭에 글을 남겨주세요.
* **Pull Request (PR)**  
    직접 내용을 수정하거나 새로운 학습 콘텐츠를 추가하고 싶다면 PR을 보내주세요. 
* **Discussions 참여**  
    커리큘럼의 방향성이나 특정 개념에 대한 토론, 새로운 주제 제안은 [Discussions] 공간을 활용해 주세요.
* **Feedback**  
    학습자 입장에서 이해하기 어렵거나 더 자세한 설명이 필요한 부분에 대한 의견도 언제나 환영합니다.
